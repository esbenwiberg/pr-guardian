How it works

  • Anthropic (Direct) sends requests to the Anthropic API. Requires an Anthropic API key.
  • Azure AI Foundry routes requests through your Azure endpoint using the Anthropic SDK's base URL parameter. Same Claude models, different billing path.
  • Settings saved here override environment variables. On restart, DB-stored settings are re-applied automatically.
  • API keys are stored in the database and never returned in full via the API.
